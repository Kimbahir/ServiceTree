Instructions:

Service tree label is shown on the actual service tree as the user friendly name

Service tree name is never shown to end-users, but is available as an internal identificator

Customer ID is never shown to end-users, but is available as an internal identificator

ITSM deeplink is for allowing direct lookup into an ITSM system. The section before %SERVER-ID% is the first part of the link and the section will be appended.

Example 1:
The server id is 1234, and the direct lookup into the ITSM system is the following:
https://myitsm.com?serverid=1234?server=true
Then the first section should be "https://myitsm.com?serverid=" and the second part be "?server=true"

Example 2:
The server id is 1234, and the direct lookup into the ITSM system is the following:
https://myitsm.com/server/1234
Then the first section should be "https://myitsm.com/server/" and the second part be "" because it is empty
