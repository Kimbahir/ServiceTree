Welcome to ServiceTree

Instructions on use:

ServiceTree is made for mapping services and their relation, without having to modify the underlying ITSM systems. The only data that are outside ITSM systems are the relations, and they will reside in the json file. The output can be returned as a full json file and/or a PDF file.

Load: Load a json input into the text area or load an example. Note, until you press submit, nothing happens :)

Patch: Load a CSV content, for when you need to update the servers and/or services. Note, it will overwrite current server registrations!

Relate: Create relations between services, by selecting two services and attaching them

Detach: Select a relation and remove said relation

View: Downloads the PDF representation of the servicetree

Persist: Downloads the JSON representation of the servicetree
