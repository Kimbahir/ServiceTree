Enter the services and servers that will replace the current services and servers, with comma separated input. Remember to sort by services

Format is: [service],[server id],[server name]

Instructions:
You can add a service list in the form of comma separated input. It can be either as a list of services only. Example:
  • Service A
  • Service B
  • Service C
Or as a list containing both Service Name, Server ID and Server Name. Example:
  • Service A,1111,Web server A
  • Service A,1112,Web server B
  • Service A,1113,Web server C
  • Service B,1211,Database A
  • Service B,1212,Database B
  • Service C,1311,Fileservices
And finally a service type can be added. The meaningful forms are:
  • farm
  • aacluster
  • apcluster
Example:
  • Service A,1111,Web server A,farm
  • Service A,1112,Web server B,farm
  • Service A,1113,Web server C,farm
  • Service B,1211,Database A,apcluster
  • Service B,1212,Database B,apcluster
  • Service C,1311,Fileservices,aacluster
  • Service C,1312,Fileservices,aacluster
  • Service D,1411,Other,other
